Native APIs

The GoAhead Native APIs consist of C language headers, function prototypes, structure definitions and defines. They are divided into one of these areas.

Area Description

Stability Classifications

Rather than providing a blanket Backwards Compatibility guarantee, APIs are individually classified according to their stability. These Stability Classifications should assist developers in migrating to new versions of GoAhead.
